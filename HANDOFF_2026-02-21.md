  ```sql
  CREATE TABLE IF NOT EXISTS task_dependencies (
      id INTEGER PRIMARY KEY AUTOINCREMENT,
      project_name TEXT NOT NULL,
      predecessor_id INTEGER NOT NULL,
      successor_id INTEGER NOT NULL,
      created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
  )
  ```

#### Auto-Detection Logic (runs at Excel import time only)
- In `server.py` inside `api_upload_excel()`: after tasks are saved, detect Finish-to-Start dependencies
- Logic: if `task_B.start_date == task_A.end_date` → they are linked (A is predecessor of B)
- Stored permanently, NOT recalculated on every page load
- **Current stored counts**: Saturn Project = 13 deps, 628MP-F31 = 31 deps

#### API Endpoints (in `server.py`)
- `GET /api/project/<name>/dependencies` → returns list of `{id, project_name, predecessor_id, successor_id}`
- `POST /api/project/<name>/dependencies` → body: `{predecessor_id, successor_id}` → creates dependency
- `DELETE /api/dependency/<id>` → deletes one dependency by ID
- `DELETE /api/project/<name>/dependencies/by-tasks` → body: `{predecessor_id, successor_id}`

## Data Model Notes

### Tasks Table Columns
`id, project_id, reference_id, name, phase, owner, start_date, end_date, status, date_closed, result, critical, milestone, row_order, created_at, updated_at`

### JS Task Object (in dashboard.html `tasks` array)
- `task.id` → integer DB id
- `task.entered` → maps to `start_date`
- `task.due` → maps to `end_date`
- `task.phase` → phase/gate label
- `task.milestone` → boolean
- `task.name` → task name

### Gate Detection (JS)
- `task.name.toLowerCase().includes('gate') && task.milestone === true`

### Dependencies Array (global in dashboard.html)
- Loaded in `loadFromAPI()` via `GET /api/project/<name>/dependencies`
- Format: `[{id, project_name, predecessor_id, successor_id}, ...]`
- Used by: `populateDepPicker`, `renderDependencyArrows`, `checkAndShowCascadePreview`, `saveTask`

## Excel Import Notes
- Excel parser: `excel_parser.py`
- Reads from row 11 onwards
- Columns: A=ref_id, B=name, C=phase, D=owner, E=start_date, F=status, G=end_date, H=date_closed, I=result
- **No predecessors column in Excel** — dependencies come only from auto-detection + manual picker
- Project name from cell C3, manager from C5
- Gate milestones: rows matching `^Gate\s*\d+\s*$` regex
